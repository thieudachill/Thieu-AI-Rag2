🛠️ Troubleshooting Guide

Common Issues:

  • CORS Error: Browser blocking requests due to missing headers
  • 500 Error: n8n webhook configuration or execution issue
  • Network Error: ngrok tunnel may be down

Quick Fixes:

  • ✅ Check n8n workflow is active and saved
  • ✅ Verify webhook URL matches exactly
  • ✅ Test webhook directly in n8n with test data
  • ✅ Restart ngrok tunnel if needed

For CORS issues, add this to your n8n HTTP Response node:

{
  "Access-Control-Allow-Origin": "*",
  "Access-Control-Allow-Methods": "POST, OPTIONS",
  "Access-Control-Allow-Headers": "Content-Type"
}
